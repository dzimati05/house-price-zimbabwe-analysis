# Zimbabwe Real Estate Analytics & House Price Prediction

## Project Overview
This project analyzes housing data and builds machine learning models to predict house prices.

The project demonstrates:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Regression Modeling
- Data Visualization
- Business Insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
